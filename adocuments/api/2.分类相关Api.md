## 1.获取分类列表<分页>
### URL
```
/font/site/api/common/type/find/pages
```

### 请求方式
```
POST
```

### 入参：
```
{
    "page":"",//页数，从0开始
    "size":"",//大小个数，大于0
    "sortType":"",//排序字段，可以为空，为空是默认排序
    "sortDirecion":"",//排序方式，0-升序DESC，1-降序SEC
    "data":{
        "level":"";//类型级别,1-一级，2-二级
    }//附加查询值
}
```


### 出参：
```
{
    "code":"",//处理代码，1001-成功，1002-失败，1003-超时
    "msg":"",//处理消息，用于提示用户
    "data":{
        "nowPage":"",//当前页数
        "totalSize":"",//总数据条数
        "data":[
            {
                "id":"";//类型id
                "name":"";//类型名称
                "parentId":"";//父类型ID，level==2时不为空
                "level":"";//类型级别,1-一级，2-二级
            },
            ...
            {
                "id":"";//类型id
                "name":"";//类型名称
                "parentId":"";//父类型ID，level==2时不为空
                "level":"";//类型级别,1-一级，2-二级
            },
        ]
    }
}
```
